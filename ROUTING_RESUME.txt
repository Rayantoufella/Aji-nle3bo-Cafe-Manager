✅ COMMENT CA MARCHE

1. Utilisateur fait une requête : GET /Aji-nle3bo-Cafe-Manager/games

2. .htaccess redirige vers :
   public/index.php?url=games

3. public/index.php :
   ✓ Charge autoload
   ✓ Définit APP_ROOT et BASE_URL
   ✓ Require router/routes.php
   ✓ Appelle router->run()

4. router->run() :
   ✓ Récupère la méthode (GET/POST)
   ✓ Récupère le chemin (/games)
   ✓ Cherche dans les routes enregistrées
   ✓ Exécute le callback

5. Callback exécuté :
   ✓ Parse la chaîne 'GameController@index'
   ✓ Cherche App\Controllers\GameController
   ✓ Crée une instance
   ✓ Appelle ->index()

6. Contrôleur répond :
   ✓ Récupère les données
   ✓ Charge la vue
   ✓ Affiche le résultat

📝 NOTES

• Namespaces acceptés :
  - App\Controller\*
  - App\Controllers\*

• Paramètres dans les routes : {id}, {slug}, {name}, etc.

• Tous les contrôleurs doivent être en App\Controller ou App\Controllers

• Chaque contrôleur doit avoir un __construct() sans paramètres obligatoires

• Les méthodes reçoivent les paramètres dans l'ordre : show($id)
